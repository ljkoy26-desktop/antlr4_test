# 목표
아래 SQL Server용 버튼 핸들러를 참고해서, DB2용 동일 기능 핸들러를 구현해줘.
로직 구조는 유지하고 렉서/파서 클래스명만 DB2용으로 교체하면 돼.

# 신규 함수명
afx_msg void OnBnClickedButtonMultiParseDB2();   // 복합 쿼리 파싱
afx_msg void OnBnClickedButtonTokenizeDB2();     // 토큰화

# 수정 대상
- 헤더: afx_msg 선언 추가
- 구현부: 핸들러 본문 작성
- 메시지 맵: ON_BN_CLICKED 매크로 추가
